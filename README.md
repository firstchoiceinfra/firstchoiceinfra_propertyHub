# FirstChoice Infra PropertyHub

A modern real estate platform for buying, selling, renting, and managing properties.

## Vision

Build a professional real estate platform similar to 99acres and MagicBricks with modern UI, secure authentication, and powerful admin management.

## Features

- Buyer Login (Mobile OTP)
- Owner Login (Mobile OTP)
- Agent Login
- Builder Login
- Secure Admin Login
- Property Upload
- Multiple Image Upload
- Video Upload
- Property Approval System
- Advanced Search & Filters
- Buyer & Seller Dashboard
- Admin Dashboard
- Notifications
- Reports & Analytics

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS

### Backend
- FastAPI
- Python

### Database
- PostgreSQL

### Storage
- Cloudinary / AWS S3
